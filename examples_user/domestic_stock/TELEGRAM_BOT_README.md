# 📱 주식 분석 텔레그램 봇

텔레그램으로 종목명을 보내면 자동으로 2년치 데이터를 분석해서 결과를 보내주는 봇입니다.

## 🚀 시작하기

### 1. 텔레그램 봇 토큰 발급

1. 텔레그램에서 [@BotFather](https://t.me/botfather) 검색
2. `/newbot` 명령어 입력
3. 봇 이름 설정 (예: MyStockAnalyzer)
4. 봇 사용자명 설정 (예: my_stock_analyzer_bot)
5. 발급받은 **토큰**을 복사 (예: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. 환경 변수 설정

```bash
# Linux/Mac
export TELEGRAM_BOT_TOKEN="여기에_발급받은_토큰_입력"

# 또는 .env 파일 생성
echo 'TELEGRAM_BOT_TOKEN="여기에_발급받은_토큰_입력"' > .env
```

### 3. 봇 실행

```bash
cd examples_user/domestic_stock
uv run telegram_bot.py
```

### 4. 텔레그램에서 봇 사용

1. 텔레그램에서 봇 검색 (위에서 설정한 사용자명)
2. `/start` 명령어로 시작
3. 종목명이나 종목코드 입력
   - 예: `삼성전자`
   - 예: `NAVER`
   - 예: `005930`

## 📊 분석 내용

봇이 제공하는 분석 정보:

### 1. 상방 거래량 분석
- 현재가보다 높은 가격대에서 거래량이 많이 터진 날
- 거래량 최대일과 현재가 근처 고거래량일 정보

### 2. 하방 거래량 분석
- 현재가보다 낮은 가격대에서 거래량이 많이 터진 날
- 거래량 최대일과 현재가 근처 고거래량일 정보

### 3. 이동평균선 분석
- MA_5, MA_10, MA_60, MA_120
- 현재가와의 차이(%) 표시
- 하방 지지선 정보

## 🛠️ 트러블슈팅

### 봇이 응답하지 않는 경우
1. 환경 변수가 제대로 설정되었는지 확인
2. 봇 실행 터미널에서 에러 로그 확인
3. KIS API 인증 정보 확인 (`kis_devlp.yaml`)

### "종목을 찾을 수 없습니다" 에러
1. 코스피/코스닥 마스터 파일 생성 필요
```bash
cd ../../stocks_info
uv run kis_kospi_code_mst.py
uv run kis_kosdaq_code_mst.py
```

### 분석이 너무 느린 경우
- API 호출 제한으로 인해 10-30초 정도 소요됩니다
- 정상적인 동작입니다

## 📝 명령어

- `/start` - 봇 시작
- `/help` - 도움말 보기
- 종목명 또는 종목코드 입력 - 분석 시작

## 🔒 보안 주의사항

- 텔레그램 봇 토큰은 절대 공개하지 마세요
- `.env` 파일은 `.gitignore`에 추가하세요
- KIS API 키도 절대 공개하지 마세요

## 💡 팁

- 여러 종목을 연속으로 요청 가능
- 분석 결과는 HTML 포맷으로 보기 좋게 표시됩니다
